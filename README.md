# pomo

`pomo` - a pomodoro timer TUI.

# Features

- Minimalistic notification (with herbe).
- No sounds.
- No session continuation.
- No notepad.
- **Only timer with settings**.

# Dependencies

- `ncurses`
- [`herbe`](https://github.com/dudik/herbe) (for notification)

# Installation

```bash
make install
```
